Disciplina Web II

Desenvolvimento de sites em HTML5, CSS e PHP e hospedagem de sites.

HTML - Linguagem de Programação

HyperText Markup Language, ou, Linguagem de marcação de Hipertexto. É o componente base de um site web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteudos.

PHP - Linguagem de Programação

PHP (originalmente chamada de Personal Home Page) é uma linguagem interpretada livre, usada originalmente apenas para o desenvolvimento de aplicações presentes e atuantes no lado do servidor, capazes de gerar conteúdo dinâmico na World Wide Web.

CSS - Linguagem de Programação

CSS (Cascading Style Sheet/Folha em Estilo Cascata) é o código que você usa para dar estilo à sua página Web.

MySQL - Banco de Dados

O MySQL é um sistema de gerenciamento de banco de dados, que utiliza a linguagem SQL como interface.
